CattyCare is a growing home-based cat daycare service. The owner wants to have a
desktop application to manage cat daycare data more efficiently. The application will store
daycare information and automatically calculate the total cost based on the duration of stay.

Main Features of the Software :
1. Cat Daycare Data Management
- The application must support adding (Create), editing (Update), and deleting (Delete)
daycare data.
- The recorded data includes : Owner Name, Cat Name, Phone Number, Duration of
Stay (Days)
- When a row in the table is clicked, the selected data must automatically appear in the
input fields (text fields) for the edit or delete process.
2. Displaying Daycare Data
- The application must display daycare data in a table format.
- The application must display the total daycare cost calculated automatically based on
the duration of stay.
- The total cost must not be entered manually. The daycare cost calculation in
CattyCare is determined based on the duration of stay :

➢ If the duration of stay is less than or equal to 2 days, the cost is Rp40,000 per day.

➢ If the duration of stay is more than 2 days, the first 2 days are still charged
      Rp40,000 per day, while the following days are charged Rp30,000 per day.
